## 📋 项目概述

这是一个基于 Node.js + Express 的**本地文件服务器**，主要用于本地文件托管、浏览和共享。项目名称为 "WebFolder"，版本 1.0.0。

---

## 🔧 技术栈

| 组件 | 版本 | 用途 |
| --- | --- | --- |
| **Express** | 5.1.0 | Web 服务器框架 |
| **Socket.IO** | 4.7.5 | WebSocket 实时通信 |
| **EJS** | 3.1.10 | 模板引擎 |
| **Multer** | 1.4.5-lts.1 | 文件上传处理 |
| **CORS** | 2.8.5 | 跨域资源共享 |
| **TypeScript** | 5.8.3 | 类型安全开发 |
| **Bootstrap** | 5.3.0 | 前端 UI 框架 |

---

## ✨ 主要功能模块

### 1️⃣ 文件浏览系统

- ✅ 目录树导航（支持多级目录）
- ✅ 文件列表展示（名称、大小、修改时间）
- ✅ 面包屑导航路径
- ✅ 文件图标识别（图片、音频、视频、文档等）
- ✅ 隐藏文件过滤（以 `.` 开头的文件）
- ✅ 安全路径检查（防止目录遍历攻击）
- ✅ 目录优先排序（目录排在文件前面）

### 2️⃣ 文件上传功能

- ✅ 支持中文文件名（UTF-8 编码处理）
- ✅ 上传到指定目录
- ✅ 文件存在性检查（覆盖前确认）
- ✅ 保留原始文件名
- ✅ 上传进度显示

### 3️⃣ 实时通信（WebSocket）

- ✅ WebSocket 连接管理
- ✅ 客户端消息回显
- ✅ 文件列表实时推送
- ✅ 特定目录内容查询
- ✅ 文件内容读取
- ✅ 连接状态监控

### 4️⃣ 留言板系统

- ✅ 匿名或昵称留言
- ✅ 留言历史存储（最多 100 条）
- ✅ WebSocket 实时推送新留言
- ✅ 时间戳记录
- ✅ 实时更新留言列表

### 5️⃣ SSE (Server-Sent Events)

- ✅ 实时数据推送
- ✅ 每秒发送时间戳（测试用）

## 🔌 API 端点

| 方法 | 路径 | 功能 | 参数 |
| --- | --- | --- | --- |
| GET | `/` | 文件列表首页 | - |
| GET | `/api/browse` | 目录浏览 API | `path`（可选） |
| GET | `/api/check-file` | 检查文件是否存在 | `path`, `name` |
| GET | `/api/messages` | 获取留言列表 | - |
| POST | `/api/messages` | 提交新留言 | `name`, `content` |
| POST | `/upload` | 文件上传 | `file`, `path`（可选） |
| GET | `/sse` | SSE 数据流 | - |

## 📁 项目结构

```
WebFolder/
├── data/              # 数据存储目录
│   └── messages.json  # 留言数据
├── public/            # 静态文件目录
│   ├── audio/         # 音频文件
│   ├── images/        # 图片文件
│   ├── resources/     # 资源文件
│   └── video/         # 视频文件
├── views/             # 模板文件
│   └── index.ejs     # 主页模板
├── index.ts          # 服务端入口
├── package.json      # 项目配置
├── tsconfig.json     # TypeScript 配置
└── pnpm-lock.yaml   # 依赖锁定文件
```

## 💡 使用场景

1. **本地文件共享**：在局域网内共享文件
